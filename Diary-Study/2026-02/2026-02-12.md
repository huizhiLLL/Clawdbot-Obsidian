## 主题：Vue 组件基础

## 核心知识点

### 1. 组件的定义

组件是 Vue 中可复用的代码块，它将 HTML、CSS 和 JavaScript 封装在一起。

**基本组件定义：**
```vue
<!-- MyButton.vue -->
<template>
  <button class="btn" @click="handleClick">
    {{ text }}
  </button>
</template>

<script setup>
defineProps({
  text: {
    type: String,
    default: 'Click me'
  }
})

const emit = defineEmits(['click'])

function handleClick() {
  emit('click')
}
</script>

<style scoped>
.btn {
  padding: 10px 20px;
  background-color: #42b983;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.btn:hover {
  background-color: #3aa876;
}
</style>
```

### 2. 组件的使用

在父组件中引入和使用组件：
```vue
<template>
  <div>
    <MyButton text="提交表单" @click="handleSubmit" />
    <MyButton text="取消" @click="handleCancel" />
  </div>
</template>

<script setup>
import MyButton from './MyButton.vue'

function handleSubmit() {
  console.log('提交表单')
}

function handleCancel() {
  console.log('取消操作')
}
</script>
```

### 3. 组件的命名

**组件文件名：**
- 推荐使用 PascalCase（大驼峰）：`MyButton.vue`
- 或使用 kebab-case（短横线）：`my-button.vue`

**组件标签：**
- 在模板中使用时，可以有两种写法：
  - `<MyButton />`（自闭合）
  - `<my-button></my-button>`（短横线，更符合 HTML 规范）

Vue 会自动转换这两种命名方式。

### 4. 组件的根元素

**Vue 2 的限制：**
- 每个组件必须有一个唯一的根元素

```vue
<!-- Vue 2 错误写法 -->
<template>
  <div>内容 1</div>
  <div>内容 2</div>  <!-- 错误：多个根元素 -->
</template>

<!-- Vue 2 正确写法 -->
<template>
  <div>
    <div>内容 1</div>
    <div>内容 2</div>
  </div>
</template>
```

**Vue 3 的改进：**
- 支持多个根元素（Fragment）

```vue
<!-- Vue 3 可以这样写 -->
<template>
  <div>内容 1</div>
  <div>内容 2</div>
</template>
```

## 概念关联

### 组件与 DOM
- 组件是模板的高级抽象
- 一个组件最终会渲染成 DOM 树中的某个节点
- 组件可以嵌套，形成组件树

### 组件与数据传递
- 父组件通过 props 向子组件传递数据（父 → 子）
- 子组件通过 emit 向父组件传递事件（子 → 父）
- 这种数据流是单向的，易于理解和管理

### 组件与复用
- 组件的核心价值是复用
- 通过 props 可以让同一个组件呈现不同的状态
- 通过插槽可以让组件内容更灵活

### 相关概念
- **props**: 父组件向子组件传递数据的方式
- **emit**: 子组件向父组件发送事件的方式
- **插槽 (Slots)**: 让组件内容更加灵活
- **生命周期钩子**: 组件在创建、更新、销毁时的回调函数
