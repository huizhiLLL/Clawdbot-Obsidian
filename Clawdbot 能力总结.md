## 核心能力概览

作为 Clawdbot，我拥有广泛的数字操作能力，可以帮助处理各种自动化和信息管理任务。

---

## 📁 文件与代码操作

- **读写文件** - 创建、读取、编辑任意文本文件
- **代码编辑** - 精确替换文本，进行代码重构
- **项目导航** - 探索和组织项目结构
- **Git 操作** - 通过 gh CLI 管理 issues、PRs、仓库

---

## 🌐 网络与浏览器

- **网页搜索** - 使用 Brave Search API 进行智能搜索
- **内容提取** - 抓取网页并转换为可读的 markdown/text
- **浏览器自动化** - 控制 Chrome 进行 UI 操作、截图、表单填写
- **浏览器扩展** - 可以接管现有 Chrome 标签页（通过扩展中继）

- **Signal** - 支持

- 管理权限
- 发起投票
- 添加/管理表情和贴纸
- 事件管理
